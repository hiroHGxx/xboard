XBoard 埋め込みデモ

1. 基本的な埋め込み

コードを表示 
<div id="xboard-basic"></div>
<script src="/xboard-embed.js"></script>
<script>
  XBoard.create('#xboard-basic', {
    accounts: 'elonmusk,twitter',
    limit: 4
  });
</script>

2. データ属性での自動初期化

コードを表示 
<div data-xboard 
     data-accounts="elonmusk" 
     data-limit="3" 
     data-theme="light"
     data-layout="column">
</div>
<script src="/xboard-embed.js"></script>

3. ダークテーマ

コードを表示 
<div id="xboard-dark"></div>
<script>
  XBoard.create('#xboard-dark', {
    accounts: 'twitter',
    limit: 2,
    theme: 'dark',
    maxColumns: 2
  });
</script>

4. カスタム設定

コードを表示 
<div id="xboard-custom"></div>
<script>
  XBoard.create('#xboard-custom', {
    accounts: 'elonmusk,twitter',
    limit: 6,
    theme: 'auto',
    layout: 'grid',
    maxColumns: 3,
    refreshInterval: 60000 // 1分
  });
</script>

📖 使用方法

1. スクリプト読み込み:

<script src="https://your-domain.com/xboard-embed.js"></script>

2. HTMLに埋め込み:

<div id="my-timeline"></div>

3. 初期化:

XBoard.create('#my-timeline', { accounts: 'elonmusk' });

⚙️ オプション一覧

オプション デフォルト 説明
accounts elonmusk,twitter 表示するXアカウント（カンマ区切り）
limit 10 表示するツイート数（1-50）
theme auto テーマ（light/dark/auto）
layout grid レイアウト（grid/column）
maxColumns 3 最大カラム数（1-5）
refreshInterval 30000 自動更新間隔（ミリ秒、0で無効）
